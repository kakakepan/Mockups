This command triggers multiple processes: It verifies the domain name's availability at the registry, generates contact handles within the registry, and initiates the registration process at the registry. There are two ways to register a new domain.

# Accessing through dashboard

 To register a new domain through dashboard, please follow the steps outlined below:
 1. Navigate to the homepage and locate the **"Online Forms"** option.
2. Click on the **"Online Forms button"** to proceed.
![6](/images/domainimages/6.png)

3. Select on the **"Domains"** by clicking on it.
![7](/images/domainimages/7.png)

4. Select the **"Register"** option to proceed.
![8](/images/domainimages/8.png)

5. Type in the **"Domain name"** that you want to register.

![9](/images/domainimages/9.png)

6. Click on the **"Search"** button to initiate the domain registration process.

<Note>üìò Note: This will generate an error code if the domain name has already been registered or is not unique.</Note>

![10](/images/domainimages/10.png)

7. Fill in all the required fields in step 2.

| Field       | Description                                                                                      |
| :---------- | :----------------------------------------------------------------------------------------------- |
| Years       | Specify the desired registration period in years for the domain.                        |
| Contact     | Specify the contact details of the domain owner.                                                 |
| DNS Servers | Specify the DNS of the domain. You can use your custom DNS or use the SuperReseller DNS servers. |

![11](/images/domainimages/11.png)

8. Click **"Order"** to proceed to the next step.

![12](/images/domainimages/12.png)

# Accessing through API
To register a new domain through API, please follow the steps outlined below:

1. Navigate to the homepage and locate the **"Knowledge base"** option.

![1](/images/contactimages/1.png)

2. Click on the **"Commands"** feature under the **"Knowledge Base"** option to proceed.

![2](/images/contactimages/2.png)

3. Select the **"Access"** option for **"Superreseller"** to proceed with registering the domain via the API.

![44](/images/domainimages/44.png)

4. You will be redirected to the API page.

![45](/images/domainimages/45.png)
