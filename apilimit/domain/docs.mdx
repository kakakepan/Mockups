The `AddDocsToPendingRequests` endpoint is an API endpoint used to add documents to a pending request. It allows you to upload and associate documents with a specific request, such as a domain registration or transfer request.

## Parameters
The following is a list of parameters for the `AddDocsToPendingRequests` endpoint:
| Parameter Name  | Data Type | Description                                                                                                       |
|-----------------|-----------|-------------------------------------------------------------------------------------------------------------------|
| cancel          | Boolean   | Determines whether to cancel the pending request. If set to `true`, the request will be cancelled.               |
| documents       | Array     | An array of document objects containing the field ID and the URL of the document to be added to the request.     |

## Example Request
The following is an example request for the `AddDocsToPendingRequests` endpoint:
```json
{
  "cancel": false,
  "documents": [
    {
      "field_id": "document_url0",
      "value": "https://document.url"
    }
  ]
}
```
