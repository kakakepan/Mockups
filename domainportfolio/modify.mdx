This command allows you to modify domain attributes such as registration periods, renewal restrictions, and status updates. It provides flexibility and control over domain management, ensuring that domains meet desired parameters and align with business needs. By leveraging this command effectively, you can optimize the performance and functionality of your domain portfolio. There are two ways to access the modify command.

## Accessing through Dashboard
To **"Modify a domain"**, follow the steps below:

1. Navigate to the homepage and locate the **"Online Forms"** option.
2. Click on the **"Online Forms button"** to proceed.
3. Click on the **"Domains"** by clicking on it.
4. Select the **"Modify"** option to proceed.
5. Type in the **"Domain"** name that you want to register.
6. Please choose the specific modify **"Command"** that you wish to apply.


## Accessing through API

To modify a domain through API, please follow the steps outlined below:

1. Navigate to the API page by clicking [here](https://superreseller-qa.nominus.host/api/swagger-ui?expand=Domains:RenewDomain#/ModifyDomai) .
2. You will be redirected to the API page.
