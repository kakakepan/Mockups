The **"AddDocsToPendingRequests"** command in the SuperReseller dashboard allows SuperResellers to add documentation to a pending domain request. It functions similarly to the **"AddFieldsToRequest"** command and validates that the entered data starts with either **"http://"** or **"https://"**. The command is located within the **"PendingRequests"** subcategory of the **"Domain"** category in Swagger.

## Access the Add Documentation Command

To Add documentation for a pending request domain, follow these steps below:
1. Navigate to the **Superreseller** dashboard here.
2. Select the **Online Form** menu.
3. Click **Add Documentation** under the Domain.

<Note>
**Note:**
The command supports character validation using regular expressions and displays validation error messages if the entered data does not pass validation.
</Note>
