This command triggers multiple processes: It verifies the domain name's availability at the registry, generates contact handles within the registry, and initiates the registration process at the registry. There are two ways to register a new domain.

# Accessing the Register Domain Command

 To register a new domain through dashboard, please follow the steps outlined below:

1. Navigate to the homepage by clicking [here](https://superreseller-qa.nominus.host/).
2. locate the **"Online Forms"** option.
3. Click on the **"Online Forms button"** to proceed.
4. Select the **"Domains"** by clicking on it.
5. Select the **Register** option to proceed.
6. Type in the domain name that you want to register.
7. Click on the **"Search"** button to initiate the domain registration process.

<Note>
**Note:**
- This will generate an error code if the domain name has already been registered or is not unique.
- If the domain is taken, it will appear in red as "**Domain is taken, cannot be registered."**
- If the domain is available, it will appear in green as **"The Domain is Available."**
</Note>

8. Fill in all the required fields.

| Field       | Description                                                                                                                                                                                           |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Years       | This field represents the possible number of years that the extension can be registered. It should be a value from 1 to 10.                                                                           |
| Reseller    | The reseller's name and ID should be displayed in this field.                                                                                                                                         |
| Price       | The price of the "Domain Registration" service, which depends on the entered extension and the selected number of years.                                                                              |
| Contacts    | In this section, select the "Contact ID" of the created contacts for the selected reseller.                                                                                                           |
| DNS Servers | Enter the DNS servers of the domain in this section. If the IP is not found, allow the user to manually enter it. The "Additional Name server" link opens a new line to enter up to 12 DNS servers.9. |

9. Click **Order** to proceed to the next step.
