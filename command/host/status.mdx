The **"Status Host"** is an informational command that allows you to check the details of a host.

<Note>
**Note:**
This command does not generate a request.
</Note>

# Accessing the Status Host Command

To access the **"Status Host"** command, follow these steps:

1. Navigate to the SuperReseller dashboard [here](https://superreseller-qa.nominus.host/).
2. Locate the **“Portfolio”** option.
3. Select the **“Hosts”** button.
4. Click on **"Status Host"** to proceed.

<Warning>
**Caution:**
- Ensure that you have the correct host name when using the **"Status Host"** command. 
- This command is solely used to check the details of a host and does not modify any data
</Warning>
