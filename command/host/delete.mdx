The **"Delete Host"** is used to delete the data of a host.

# Accessing the Delete Host Command

To access the **"Delete Host"** command, follow these steps:

1. Navigate to the SuperReseller dashboard [here](https://superreseller-qa.nominus.host/).
2. Locate the **“Portfolio”** option.
3. Select the **“Hosts”** button.
4. Click on **"Delete Host"** to proceed.

<Warning>
**Caution:**
- When using the **"Delete Host"** command, ensure that you comply with the same restrictions and guidelines that currently apply to the command. 
- Carefully reviewing the data before deletion will maintain data integrity and ensure accurate host deletions.
</Warning>
