The **"Update Host"** command is used to update the data of a host.

# Accessing the Update Host Command

To access the **"Update Host"** command, follow these steps:

1. Navigate to the SuperReseller dashboard [here](https://superreseller-qa.nominus.host/).
2. Locate the **“Portfolio”** option.
3. Select the **“Hosts”** button.
4. Click on **"UpdateHost"** to proceed.

<Warning>
**Caution:**
- When using the **"Update Host"** command, ensure that you comply with the same restrictions and guidelines that currently apply to the command.
- Providing correct and authentic information will maintain data integrity and ensure accurate host updates.
</Warning>
