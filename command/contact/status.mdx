The **"Status Contact"** command is an informational command that allows resellers to check the details of a contact. It does not generate a request.

# Accessing the Status Contact Command

To access the **"Status Contact"** command, follow these steps:

1. Navigate to the SuperReseller dashboard [here](https://superreseller-qa.nominus.host/).
2. Select the **"Online Form"** menu.
3. Click on **"Status Contact"** under the **"Contact"** section.

<Warning>
**Caution:**
- Ensure that you have the correct contact handle ID when using the **"Status Contact"** command.
- This command is solely used to check the details of a contact and does not modify any data.
</Warning>
