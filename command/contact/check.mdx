The **"Check Contact"** command is an informational command that allows you to check the availability of a contact ID (Handle ID). It does not generate a request.

# Accessing the Check Contact Command

To access the **"Check Contact"** command, follow these steps:

1. Navigate to the SuperReseller dashboard [here](https://superreseller-qa.nominus.host/).
2. Select the **"Online Form"** menu.
3. Click on **"Check Contact"** under the **"Contact"** section.

<Warning>
**Caution:**
- Ensure that you have the correct contact handle ID when using the **"Check Contact"** command.
- This command is solely used to check the availability of a contact ID and does not modify any data.
</Warning>
