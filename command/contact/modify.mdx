# Modify a Contact

The process of modifying **"contacts and commands"** of a domain involves making changes to the contact and command information associated with a particular domain name registration. This functionality allows domain owners or administrators to update and manage the contact details.


## Access the Mofify Contact Command
To access **"Modify Contact"** page using the **"Super Reseller dashboard"**, follow the steps outlined below:

1. Click the **Online Forms** button on the home page.

2. Click the **Contacts** button.

3. Select **Modify** to create a new contact.

4. Specify the **Contact ** you want to modify.

5. Specify the **Command**. You can choose to **Update** or **Delete** the Host.
