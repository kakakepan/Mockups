The **"Delete Contact"**  command is used to delete the data of a contact.

# Accessing the Delete Contact Command

To access the **"Delete Contact"** command, follow these steps:

1. Navigate to the SuperReseller dashboard [here](https://superreseller-qa.nominus.host/).
2. Select the **"Online Form"** menu.
3. Click on **"Delete Contact"** under the **"Contact"** section.
4. On the **"Modify Contact"** page,  you will find a form to delete the contact data.

<Warning>
**Caution:**
- Ensure that you have the correct contact handle ID when using the **"Check Contact"** command.
- This command is solely used to check the availability of a contact ID and does not modify any data.
</Warning>
